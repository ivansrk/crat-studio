## 1. Позиционирование

**CRAT studio — студия прикладного ИИ.** Обучаем. Автоматизируем. Создаём.
Гибрид: AI-образование · автоматизации для малого/среднего бизнеса · креативное производство (ИИ + ручная работа).

Запрещённые формулировки: «прокачай навыки», «стань экспертом за 7 дней», «под ключ» в инфостиле, «лучшие решения», «современная образовательная платформа».

### Наследование SKLAD → CRAT
- Публичный бренд — CRAT studio. SKLAD — фактура опыта, НЕ второй бренд на сайте, без истории переименования, без двух логотипов.
- Формула: «CRAT studio выросла из многолетней образовательной практики: команда с 2013 года запускает курсы, живые группы и AI-инструменты в реальных задачах».
- НЕ писать «раньше мы были SKLAD».

## 2. Референсы (не копировать буквально)
MUBI (чёрный фон, кинематографичность, минимум шума) · NOWNESS (арт-видео-журнал) · Are.na (архив/коллекции) · e-flux (редакционность, плотная типографика) · School of Motion (структура образовательного продукта) · Domestika (каталог, но темнее и взрослее).

## 3. Визуальная формула
**Black is the space. Red is the light. Image is the protagonist. Interface is quiet.**
Настроение: ночной город, красные вывески, маленький кинотеатр, архив, мастерская, ручная линия, несовершенство. Без SaaS-глянца, стоков, синих кнопок, стартап-градиентов. Красный = свет: glow, underline, hover, тонкие линии, прогресс.

## 4. Токены (расширение lib/design/tokens.css — существующие НЕ менять)
```css
--bg-black: #050505; --bg-void: #000000;
--red-blood: #B11212; --red-dark: #4A0909;
--cyan-faded: #48D6C0; --violet-shadow: #4A2A5E;

```
Правила: фон bg-deep/bg-black; текст paper; вторичный paper-muted; интерактив neon; мята/бирюза/лаванда — редкие акценты; не заливать сайт красным.

## 5. Типографика
Cormorant Garamond (заголовки) · Manrope (текст) · JetBrains Mono (label/tags, UPPERCASE для маленьких маркеров: `[STUDIO / APPLIED AI]`). Крупно, много воздуха, короткие фразы, не мельчить (UX-01 незыблем).

## 6. Анимации (CSS-only)
Разрешено: slow fade-in, image reveal, red glow pulse, subtle grain, slow zoom на визуальных блоках, underline reveal, лёгкий подъём карточки на hover, мягкое появление секций. Запрещено: bouncing, spring, параллакс ради параллакса, мультяшность. prefers-reduced-motion уже глушится глобально — сохранить. Классы: .fade-up .red-glow .frame-noise .cinematic-card .reveal-line + utility: .crat-shell .crat-section .crat-kicker .crat-display .crat-muted .crat-button (+.secondary) .crat-card .crat-grid .crat-visual-frame .crat-noise .crat-red-line .crat-tag.

## 7. Главная `/` — редакционный монтаж
1. **Header** (все публичные страницы): липкий, тихий; слева CRAT studio + подпись-моно «applied AI / education / automation / creative work»; справа nav (см. адаптации) + Войти/В кабинет. Hover: красный underline + лёгкое свечение.
2. **Hero**: label → большой заголовок → подзаголовок → CTA «Смотреть курс» (/ai-basics) и «Обсудить автоматизацию» (/#contact) → CSS visual frame (киноэкран: красное свечение, шум, неоновые линии — БЕЗ картинок).
3. **Три направления** «Что делает CRAT studio» — карточки-постеры (номер, label-моно, заголовок, текст, красная линия): Обучение / Автоматизации / Креативная студия (тексты §12).
4. **Курс в фокусе** (id=course): данные из getContent() (title; «4 модуля / 12 уроков / мини-проект / сертификат» — числа из course.yaml/lessonCount, не хардкод) → «Открыть программу» → /ai-basics.
5. **Автоматизации** (id=automation): заголовок «Автоматизации без корпоративного театра» + 4 compact-карточки (Документы и сертификаты · CRM и клиентские базы · Чат-боты и ассистенты · Рассылки, отчёты и процессы) + CTA «Обсудить задачу» → /#contact.
6. **Креативная студия** (id=studio): «ИИ + ручная работа» + асимметричная сетка 3–5 CSS-кадров (красный горизонт, зелёное пятно, неоновая линия, зерно).
7. **Этика**: «Мы не строим автоматизации ради автоматизаций. Сначала разбираем задачу, риски и границы — потом решаем, где ИИ действительно нужен, а где лучше оставить ручную работу. ИИ не заменяет человека: он берёт на себя техническую, повторяющуюся и операционную работу — там, где это безопасно, этично и экономит время.»
8. **Как мы работаем**: 4 монтажные карточки (Разбираем задачу → Проектируем решение → Собираем прототип → Обучаем команду и передаём процесс).
9. **Команда** (id=team): заголовок «Команда, которая соединяет образование, ИИ и реальные процессы»; карточки-«архивные досье» (номер, имя, роль, описание, mono-теги, CSS-плейсхолдер вместо фото). Состав — §11, БЕЗ Валерии Таран.
10. **Опыт**: «Из онлайн-образования — в прикладной ИИ» + 4 факта: с 2013 года в онлайн-образовании · курсы и живые группы · личные кабинеты, материалы и сертификаты · прикладные AI-инструменты в реальных задачах. Чисел не выдумывать.
11. **Контакты** (id=contact): «Есть задача для курса, автоматизации или визуального проекта?» + кнопки «Оставить заявку на курс» (/ai-basics#signup) и «Обсудить проект» (mailto из словаря).
12. **Footer**: CRAT studio · Студия прикладного ИИ · Обучение / автоматизации / креативное производство · nav-ссылки · email.
